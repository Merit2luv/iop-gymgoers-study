# Data

This folder will contain the participant dataset (coded by Participant ID — no personal identifiers).

**To add:** drop in your raw and/or cleaned dataset (CSV/XLSX) here, e.g.:
- `iop_raw_data.xlsx` — original data collection sheet entries
- `iop_cleaned_data.csv` — cleaned dataset used for SPSS analysis

Suggested columns: `Participant_ID`, `Group` (Gym-goer/Non-gym-goer), `Age`, `Gender`, `BMI`, `Systolic_BP`, `Diastolic_BP`, `Exercise_Frequency`, `IOP_Right`, `IOP_Left`, `IOP_Avg`.
